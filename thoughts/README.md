# Shared Thoughts Guide

`thoughts/` is reserved for shared artifacts that can support multiple tutors.

## Folder roles

- `thoughts/research/` stores research notes and source summaries.
- `thoughts/plans/` stores implementation and design plans.

Tutor-specific curriculum, tracking, session, and workflow files do not belong in `thoughts/`. They live under `tutors/<tutor-name>/`.

## Current canonical split

- Shared exam scope research for AWS SAP: `thoughts/research/2026-03-28-1545-aws-solutions-architect-professional-exam-study.md`
- AWS SAP tutor operating files: `tutors/aws-solutions-architect/`

## Naming conventions

- Shared research notes use `YYYY-MM-DD-HHMM-topic.md`.
- Shared plans use `YYYY-MM-DD-topic-plan.md` or another descriptive dated plan name.
- Tutor-local session notes use `YYYY-MM-DD-topic.md`.
- Objective IDs stay stable across curriculum and tracking documents and use the form `SAP-Dx-Oy`.
